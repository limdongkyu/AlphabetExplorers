# 알파벳 탐험대 - Alphabet Explorers 🎈

초등학생 1-2학년 아이들을 위한 재미있고 인터랙티브한 알파벳 학습 웹 애플리케이션입니다.

## ✨ 주요 기능

### 📚 알파벳 배우기
- 대소문자를 함께 배울 수 있는 인터랙티브 카드
- Web Speech API를 활용한 음성 재생 기능
- 각 알파벳으로 시작하는 쉬운 단어 3-5개 학습
- 시각적 피드백과 애니메이션 효과

### ✍️ 쓰기 연습하기
- Canvas 기반 손글씨 연습 기능
- 터치 및 마우스 입력 지원
- 대소문자 선택 가능
- 연습 횟수 추적 및 완료 표시

### 🎯 단어 배우기
- 알파벳별 단어 카드 모음
- 이모지를 활용한 시각적 학습
- 영어 단어와 한국어 뜻 함께 표시
- 단어 음성 재생 기능

### 🌟 성취도 추적
- LocalStorage를 활용한 학습 진행 상황 저장
- 별점 시스템 (0-3점)
- 완료한 알파벳 및 쓰기 연습 추적
- 전체 진행률 시각화

## 🚀 시작하기

### 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 🛠️ 기술 스택

- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **음성 기능**: Web Speech API (TTS)
- **그리기 기능**: HTML5 Canvas API
- **데이터 저장**: LocalStorage

## 📱 사용 방법

1. **메인 페이지**: 홈에서 원하는 학습 메뉴 선택
2. **알파벳 배우기**: 알파벳 카드를 클릭하여 대소문자와 단어 학습
3. **쓰기 연습**: 화면을 그려서 알파벳 쓰기 연습
4. **단어 배우기**: 알파벳을 선택하고 관련 단어들을 배움
5. **성취 보기**: 학습 진행 상황과 별점 확인

## 🎨 디자인 특징

- 아이들 친화적인 밝고 화사한 색상
- 큰 버튼과 텍스트로 사용성 최적화
- 반응형 디자인 (모바일, 태블릿, 데스크톱 지원)
- 부드러운 애니메이션과 시각적 피드백

## 💡 학습 목표

- 알파벳 대소문자 명확하게 구분하기
- 알파벳 소리내어 읽기
- 각 글자로 시작하는 쉬운 단어 3-5개 학습
- 직접 쓰기 연습을 통한 손으로 익히기
